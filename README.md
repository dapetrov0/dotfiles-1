# dotfiles

My dotfiles.

## Installation

```sh
cd ~
git clone https://github.com/jaredpalmer/dotfiles.git && cd dotfiles && ./sync.sh
```

To update later on, _after you push to github_, just run the ./sync again. Note: `.extra` is not committed.
